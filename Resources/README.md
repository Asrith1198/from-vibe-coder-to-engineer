# 📚 Curated Resources & Links

> "Stand on the shoulders of giants."

A structured directory containing high-quality resources, books, documentation, interactive sites, and tools to help accelerate the journey from vibe coding to real engineering.

## 🐍 Python Resources
- [Python Official Documentation](https://docs.python.org/3/) — The ultimate authority.
- [Python 101 by Real Python](https://realpython.com/) — Clean, practical, high-quality Python tutorials.
- [Python Playground](https://github.com/geekcomputers/Python) — Community scripts to learn from.

## 🧠 DSA (Data Structures & Algorithms) Resources
- [LeetCode](https://leetcode.com/) — Practice problems, community discussions.
- [NeetCode.io](https://neetcode.io/) — Structured roadmap with detailed video solutions.
- [GeeksforGeeks DSA](https://www.geeksforgeeks.org/data-structures/) — Concepts explained with step-by-step code.
- [Visualgo.net](https://visualgo.net/en) — Interactive algorithm visualizations.

## 🤖 AI & Machine Learning Resources
- [Kaggle Learn](https://www.kaggle.com/learn) — Practical micro-courses on Pandas, ML, Deep Learning.
- [Fast.ai](https://www.fast.ai/) — Outstanding, top-down deep learning courses.
- [StatQuest with Josh Starmer (YouTube)](https://www.youtube.com/@statquest) — Simplifies machine learning math beautifully.

## 🚀 Startup & Scale Resources
- [Y Combinator Startup School](https://www.startupschool.org/) — Essential training for solo founders.
- [System Design Primer](https://github.com/donnemartin/system-design-primer) — Outstanding guide on how to scale systems.
