# 🤖 AI & Machine Learning

> "The details of how neural networks learn are simple. The details of how they are applied are complex."

A home for exploring data science, statistics, model building, and deep learning, building a strong foundation in modern AI.

## 🗺️ Roadmap & Curriculum

### 1. Data Manipulation & Analysis
- [ ] **NumPy**: Multidimensional arrays, linear algebra, vectorization.
- [ ] **Pandas**: DataFrames, series, reading CSV/databases, data cleaning, and filtering.
- [ ] **Matplotlib & Seaborn**: Creating insightful visualizations and analyzing distributions.

### 2. Machine Learning Fundamentals
- [ ] **Regression**: Linear and Logistic Regression.
- [ ] **Classification**: Decision Trees, Random Forests, SVMs, Naive Bayes.
- [ ] **Clustering & Dimensionality Reduction**: K-Means, PCA.
- [ ] **Model Evaluation**: Overfitting/Underfitting, Cross-validation, Precision/Recall, ROC-AUC.

### 3. Deep Learning & Beyond
- [ ] **Neural Networks**: Feedforward networks, activation functions, backpropagation.
- [ ] **Frameworks**: PyTorch or TensorFlow.
